# SpoilerFreePlexSports

A Docker-based companion service for Plex that watches for new sports recordings,
identifies the game (Gemini + TheSportsDB), and organizes each game into its own
folder with **spoiler-free** artwork and metadata — so Plex never shows you a
thumbnail with the score on it.

**Status:** design phase. See [design.md](design.md) for the full architecture and
phased build plan. Nothing is built yet.
